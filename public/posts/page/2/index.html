crop pic in current layer

ps中如何裁剪当前图层的图像，而不影响其他

使用反向选择然后删除

  1. 选中当前图层（在“图层”面板中点击该图层）。
  2. 使用选框工具（快捷键：M），选择你想保留的部分。
  3. 执行“选择”→“反向”（Shift + Ctrl + I / Shift + Command + I），反选多余部分。
  4. 按 Delete 删除 选中部分（适用于像素图层）。

使用PS Mask

  1. 添加矩形选区：使用“矩形选框工具”（M）绘制选区。
  2. 新建一个“矢量蒙版”：在“图层”面板底部点击“添加蒙版”按钮。
  3. 调整蒙版：如果需要修改裁剪区域，可以点击蒙版缩略图并调整。

这种方法不会永久删除内容，随时可以调整或恢复。

Emmet

  • belong: TODO
  • relation: TODO

https://docs.emmet.io/

Emmet是一个插件,用于类似Emacs,vs code等文本编辑器中 Emmet 是一套快速编写 HTML/CSS 的缩写语法与扩展工具，极大提高了前端开发效率。

它解决的问题：

  • HTML/CSS 编写冗长、重复，效率低下
  • 结构嵌套复杂、手写费时
  • 缺乏一致性的手动缩进与格式化

它用于提高编写 HTML&CSS 的效率

比如编写

div>ul>li

然后执行 Expand Abbreviation (一般是 Tab 快捷键)它会自动扩展成

<div>
    <ul>
        <li></li>
    </ul>
</div>

所以它有一套 Abbreviations Syntax 是要学的
