`C-b %` split horizontal

`C-b "` split vertical

`C-b x` close current pane

`C-b t` show clock in current pane

`C-b o` select next pane

`C-b <arrow>` select the pane that arrow indicated

`C-b {` swap the current pane with the previous one

`C-b }` swap the current pane with the next one

`C-b z` maximizes the current pane to full-screen
