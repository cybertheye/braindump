可以给一个node添加一个ref,
这个ref可以是

-   url
-   [org-ref]({{< relref "20250323220733-org_ref.md" >}}) citation key

相当于这个节点,绑定了这个ref

如果有其他 node 使用了这个 url 或者同样的 org-ref citation key
那么就会在绑定ref node 的 org-roam buffer 以 Reflink 形式(其实也是backlink) 中出现, 比如当前这个 node, 现在使用 `org-roam-buffer-toggle` 试试看

方法:

-   `org-roam-ref-add`
-   `org-roam-ref-remove`

-   relation: [org-roam]({{< relref "20250318225328-org_roam.md" >}}) [org-ref]({{< relref "20250323220733-org_ref.md" >}})
